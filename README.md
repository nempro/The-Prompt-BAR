# The Prompt BAR
「AI画像生成をもっと快適に。」
The Prompt BAR は、AI画像生成で使うプロンプト入力を、もっと快適にするために生まれたデスクトップツールです。

AI画像生成向けのシンプルなデスクトップ翻訳ツールです。

日本語と英語のプロンプトを素早く翻訳し、
コピー＆ペーストを快適に行えることを目的としています。
