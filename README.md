# The Prompt BAR

AI画像生成向けのシンプルなデスクトップ翻訳ツールです。

日本語と英語のプロンプトを素早く翻訳し、
コピー＆ペーストを快適に行えることを目的としています。

---

## 特徴

- 日本語 ⇔ 英語翻訳
- シンプルなUI
- コピー・ペースト機能
- Windows対応
